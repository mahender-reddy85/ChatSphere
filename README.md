# ChatSphere

A modern, real-time chat application built with React, TypeScript, and Firebase.

## Features

- 🚀 Real-time messaging with Firebase
- 🎨 Dark/light theme support
- 📱 Mobile-optimized responsive design
- 🔊 Notification sounds and vibration
- 🗑️ Clear chat functionality
- 👥 Online/offline presence indicators
- 🔒 Secure authentication (Google, Email, Guest)
- 📱 QR code room invitations
- ⚡ Typing indicators
- ✅ Message read receipts

## Tech Stack

- **Frontend**: React 18, TypeScript, Tailwind CSS
- **UI Components**: shadcn/ui, Lucide Icons
- **Backend**: Firebase (Firestore, Authentication, Realtime Database)
- **Build Tool**: Vite
- **Deployment**: Vercel

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up Firebase configuration
4. Start development server: `npm run dev`

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
- `npm run test` - Run tests
